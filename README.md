# Netflix Clone

A front-end clone of the Netflix landing page, built using **HTML** and **CSS** to practice responsive layout design, flexbox, and UI replication of real-world websites.

## 🛠️ Built With
- **HTML5** – page structure and content
- **CSS3** – styling, layout, and responsiveness (Flexbox, Grid)

## ✨ Features
- Full-screen hero section with background image overlay and email sign-up CTA
- "Enjoy on your TV" and "Download to watch offline" content sections with alternating image/text layout
- FAQ section with expandable-style question boxes
- Multi-column footer with quick links
- Netflix-inspired color scheme and typography

## 📂 Project Structure
```
Netflix-Clone/

    └── images/        # Logo, background, and thumbnail images
```

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/definetlyNot22/Netflix-Clone.git
   ```
2. Navigate into the project folder
   ```bash
   cd Netflix-Clone
   ```
3. Open `Netflix.html` in your browser

No build tools or dependencies required — it's pure HTML/CSS.
